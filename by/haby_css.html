Cascading Style Sheets

Cascading Style Sheets, yang akrab disebut sebagai CSS, adalah bahasa desain sederhana yang dimaksudkan untuk menyederhanakan proses pembuatan halaman web yang layak. CSS menangani tampilan dan nuansa bagian dari halaman web. Dengan menggunakan CSS, Anda dapat mengontrol warna teks, gaya font, jarak antar paragraf, bagaimana ukuran kolom dan tata letak, gambar latar atau warna apa yang digunakan, serta berbagai efek lainnya. CSS mudah dipelajari dan dipahami tetapi memberikan kontrol yang kuat atas penyajian dokumen HTML. Paling umum, CSS dikombinasikan dengan bahasa markup HTML atau XHTML.

CSS terdiri dari aturan gaya yang ditafsirkan oleh browser dan kemudian diterapkan ke elemen yang sesuai dalam dokumen Anda. Aturan gaya dibuat dari tiga bagian:
1. Selector: Selector adalah tag HTML di mana gaya akan diterapkan. Ini bisa berupa tag seperti h1 atau table dll.
2. Properti: Properti adalah jenis atribut dari tag HTML. Sederhananya, semua atribut HTML dikonversi menjadi properti CSS. Bisa warna, batas, dll.
3. Nilai: Nilai ditugaskan ke properti. Misalnya, properti warna dapat memiliki nilai merah atau # F1F1F1 dll.

Anda dapat menempatkan Sintaks Aturan Gaya CSS sebagai berikut:

selector { property : value}

Contoh: Anda dapat mendefinisikan batas tabel sebagai berikut:

table{ border :1px solid #C00; }

Di sini tabel adalah pemilih dan perbatasan adalah properti dan nilai yang diberikan 1px solid # C00 adalah nilai properti itu. Anda dapat menentukan penyeleksi dengan berbagai cara sederhana berdasarkan kenyamanan Anda. Biarkan saya meletakkan penyeleksi ini satu per satu.

The Type Selectors


Ini adalah pemilih yang sama yang telah kita lihat di atas. Sekali lagi, satu contoh lagi untuk memberi warna pada semua judul level 1:

h1 {
color: #36CFFF;
}

The Universal Selectors


Daripada memilih elemen jenis tertentu, pemilih universal cukup cocok dengan nama jenis elemen apa pun:

* {
color: #000000;
}

The Class Selectors


Anda bisa mendefinisikan aturan gaya berdasarkan atribut kelas elemen. Semua elemen yang memiliki kelas itu akan diformat sesuai dengan aturan yang ditetapkan.

.black {
color: #000000;
}

The ID Selectors


Anda bisa mendefinisikan aturan gaya berdasarkan atribut id elemen. Semua elemen yang memiliki id itu akan diformat sesuai dengan aturan yang ditentukan.

#black {
color: #000000;
}
