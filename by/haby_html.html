About HTML

Secara sederhana, halaman Web (atau dokumen HTML) adalah file teks biasa yang sudah ada dikodekan menggunakan Hypertext Markup Language (HTML) sehingga tampak diformat dengan baik di browser Web. Inilah yang dimaksud dengan HTML, kata demi kata:
● Teks Hiperteks yang Anda klik untuk berpindah dari satu dokumen ke dokumen lainnya. Ini adalah referensi ke kemampuan halaman Web untuk saling terhubung satu sama lain.
● Markup Tag yang menerapkan tata letak dan konvensi pemformatan ke teks biasa. Secara harfiah, teks biasa "ditandai" dengan tag.
● Bahasa Referensi untuk fakta bahwa HTML dianggap sebagai bahasa pemrograman.

HTML umumnya diproduksi dan dikonsumsi sebagai dokumen. Kami mungkin berbicara tentang cuplikan HTML tetapi dalam kebanyakan kasus, situs web akan menanggapi permintaan dengan dokumen HTML lengkap, yang kemudian diuraikan oleh klien dan diberikan pada perangkat. Sebagai pengantar singkat tentang sintaks HTML, mari kita lihat dokumen HTML yang sangat sederhana. Listing 1-1 termasuk versi dokumen HTML Hello World. <!DOCTYPE html>
<html lang="en">
<head>
<meta charset="utf-8" />
<title>HTML5 Sample Document</title>
</head>
<body>
Hello World!
</body>
</html>
